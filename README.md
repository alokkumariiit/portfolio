# Alok Kumar | Cloud & DevOps Portfolio

A modern, responsive personal portfolio website built using **HTML5** and **CSS3**. This portfolio showcases my Cloud and DevOps journey, technical skills, production-grade projects, and career roadmap with a clean dark-themed UI.

## 🚀 Live Preview

> Open `index.html` in any modern web browser.

---

## ✨ Features

- Modern dark UI with glassmorphism effects
- Fully responsive layout
- Sticky navigation bar
- Hero landing section
- About section
- Technical skills showcase
- DevOps project portfolio
- Learning timeline
- Achievements section
- Smooth scrolling navigation
- Pure HTML & CSS (No JavaScript)

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Responsive Design
